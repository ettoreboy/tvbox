# Security

If you find a security issue (e.g. credential handling, script injection, or unsafe defaults), please report it responsibly.

**Do not** open a public issue. Instead:

- **Email** the maintainers (see the commit history or profile of the repo owner), or
- If the repo has **Security advisories** enabled: open **Security → Advisories** and create a draft advisory.

We will respond and work on a fix. Thank you.
